# 🤖 AI Fashion Prompts

Una colección de prompts de alta calidad para **ChatGPT** y otros modelos de IA enfocados en moda, estilo personal, análisis de imágenes y creación de contenido.

El objetivo de este repositorio es reunir prompts listos para usar que permitan obtener resultados más precisos, consistentes y útiles en temas relacionados con imagen personal.

## 📚 Contenido

Actualmente el repositorio incluye prompts para:

* 🎨 Colorimetría personal
* 👔 Análisis de outfits
* 🧥 Recomendaciones de estilo
* 🎯 Guías visuales e infografías
* 📸 Análisis de fotografías
* 🤖 Automatización de tareas con IA

Con el tiempo se irán agregando nuevos prompts y mejorando los existentes.

## 🚀 Cómo usar

1. Abre ChatGPT u otro modelo compatible.
2. Selecciona el prompt que quieras utilizar.
3. Copia y pega su contenido.
4. Si el prompt lo requiere, adjunta la imagen o proporciona la información solicitada.
5. Ejecuta el prompt.
